# Carpeta de entregas

Esta carpeta contiene una subcarpeta para cada estudiante donde pueden subir las soluciones de los dos talleres y del trabajo final.

Formato recomendado de archivos:
- `taller1_<ID>.<ext>` (ej. `taller1_1034918023.pdf`)
- `taller2_<ID>.<ext>`
- `proyecto_final_<ID>.<ext>` (ej. `.zip` o `.pdf`)

Cada estudiante tiene su propia carpeta nombrada con su número de cédula (`<ID>`). Para mantener la privacidad, **NO** incluimos nombres en los nombres de carpeta.

La correspondencia ID ↔ Nombre ↔ Email se guarda en `profesor/recursos_privados/mapping.csv` (carpeta privada).

Si quieres que además cree subcarpetas `taller_1`, `taller_2` y `proyecto_final` dentro de cada carpeta de estudiante, dímelo y lo creo.
