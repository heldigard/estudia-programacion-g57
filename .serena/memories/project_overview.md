# Project Overview - EstudIA-Programacion

## Purpose
Curso de introducción a la programación en Python y JavaScript básico para el programa Estud-IA / Digital X. Es un proyecto educativo que combina:
- Aprendizaje de Python fundamentals (60% virtual)
- Introducción a conceptos web con JavaScript/HTML/CSS (40% presencial/mixto)
- Material para estudiantes y recursos privados para el docente

## Target Audience
- Jóvenes (16-28) y adultos (29-59) iniciando en programación
- Modalidad mixta: virtual en Google Colab, presencial para prácticas

## Tech Stack
- **Python**: Google Colab (no requiere instalación local)
- **Web**: HTML5, CSS3, JavaScript vanilla
- **Platform**: Google Colab, CodePen, Replit
- **Version Control**: Git

## Project Structure

```
estudia-programacion/
├── estudiantes/              # Material para estudiantes
│   ├── notebooks/           # Notebooks Jupyter para Colab (.ipynb)
│   ├── datasets/            # Archivos CSV para ejercicios
│   ├── assignments/         # Tareas y evaluaciones
│   └── templates/           # Plantillas web (HTML/CSS/JS)
│       ├── web_base/        # Plantilla base
│       └── web_sessions/    # Adaptaciones por sesión (s02-s27)
├── profesor/                # Material del docente (privado)
│   ├── solutions/           # Soluciones de ejercicios
│   ├── rubrics/             # Rúbricas de evaluación
│   └── recursos_privados/   # Scripts y guías docentes
└── memory-bank/             # Documentación del proyecto (serena ignored)
```

## Session Structure
- **Virtual (S01-S12)**: Python fundamentals en Google Colab
- **Mixed (S15, S18)**: Archivos y E/S en Python
- **Mixed (S21, S24)**: Resolución de problemas técnicos
- **Mixed (S26-S27)**: Desarrollo de aplicaciones prácticas (mini-proyecto)
